# Trader Behavior vs Market Sentiment Analysis

## Project Overview
Analysis of Hyperliquid trader performance against the Bitcoin Fear & Greed Index to identify behavioral patterns.

## Folder Structure
- `notebooks/`: Contains the .ipynb analysis file.
- `csv_files/`: Contains raw data (`historical_data.csv`, `fear_greed_index.csv`).
- `outputs/`: Contains EDA visualizations (.png files).
- `ds_report.pdf`: Summary report of insights.

## Key Insight
Traders show maximum profitability ($67.89 avg) during 'Extreme Greed', despite the highest trading volumes occurring during 'Fear' phases.
